Java Card

Java Card es la plataforma abierta más importante del mundo para elementos seguros incrustados, tarjetas SIM y tarjetas inteligentes, que proporciona un entorno interoperable abierto y seguro que puede almacenar y actualizar varias aplicaciones en un único dispositivo, lo que permite el desarrollo y la implementación de servicios de identidad de confianza portátiles para individuos y dispositivos personales.

Más información

Java Cloud Service

Cree mejores aplicaciones, de manera más rápida, inteligente y eficaz.

Oracle Java Cloud Service es una plataforma de nube de nivel empresarial, basada en suscripciones, de autoservicio, confiable, escalable y elástica, que permite a las empresas desarrollar e implementar aplicaciones Java de manera segura.

Más información

Java EE

Potencie su capacidad de elección y reduzca al mínimo los riesgos; desarrolle servicios escalables para aplicaciones web y móviles.

Con diversas implementaciones comerciales y de código abierto, Java EE es el estándar empresarial y definido por la comunidad para crear servicios empresariales escalables que alimenten aplicaciones web dinámicas y móviles.

Más información

Java Embedded

Desate el valor de Internet of Things con inteligencia en los dispositivos conectados.

Desarrolle soluciones altamente funcionales, confiables, portátiles y seguras para dispositivos incorporados pequeños a grandes. Reduzca costos, mejore la calidad del producto y acelere el tiempo de salida al mercado con Oracle Java Embedded.

Más información

Java ME Embedded Client

Cree aplicaciones Java que se ejecuten en pequeños dispositivos con memoria limitada, despliegue y capacidad de energía. Oracle Java ME Embedded Client es un tiempo de ejecución de aplicaciones que se basa en la popular especificación Java ME y está diseñado para dispositivos con recursos limitados, como lectores de libros electrónicos, reproductores de discos Blu-ray, teléfonos VOIP, televisores, decodificadores, impresoras y puertas de enlace residenciales.

Más información

Java ME SDK

Oracle Java ME Software Development Kit (SDK) 8, junto con determinados plugins para IDE populares, como NetBeans, ofrece un completo entorno de desarrollo para aplicaciones incorporadas. Con estas herramientas, los desarrolladores pueden desarrollar, implementar, probar y depurar sus aplicaciones en escritorios de Windows utilizando emuladores de dispositivos incorporados y soporte integrado para el perfilado y el monitoreo de la red.

Más información

Java SE

La plataforma de desarrollo número 1

Desarrolle aplicaciones seguras, portátiles y de alto rendimiento para la más amplia gama posible de plataformas computacionales, a la vez que impulsa la productividad de usuario final y reduce drásticamente el costo de propiedad de las aplicaciones.

Más información

Java SE Embedded

Oracle Java SE Embedded es una implementación de la plataforma Java SE con una amplia variedad de funciones, optimizada para sistemas incorporados. Le permite desarrollar aplicaciones altamente funcionales, confiables y portátiles para los más potentes sistemas y dispositivos incorporados actuales.

Más información

Java TV

Java TV es una plataforma líder mundial de contenidos interactivos de televisión digital, que ofrece una plataforma madura, de alto rendimiento, dispositivos de cliente basados en televisión y medios de comunicación, y que permite el desarrollo de aplicaciones Java interactivas y atractivas para reproductores de discos Blu-ray, televisores, decodificadores y mucho más.
